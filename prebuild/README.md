配套工具使用说明
如果缺少.net请先安装dotNetFx40_Full_x86_x64
https://download.microsoft.com/download/9/5/A/95A9616B-7A37-4AF6-BC36-D6EA96C8DAAE/dotNetFx40_Full_x86_x64.exe


20171221 对外公开附加工具软件免费使用，但不包软件售后。

1.安卓板上运行的应用xuebaoRtmpPush  (仅限雪暴娃娃机配套主板，安卓板固件请升级至1221版本或以上，安卓系统版本4.4)

包含功能:源码现已对外开放。

1.1通用RTMP的低延迟推流

1.2开机自启动（留意第一次安装后，要启动应用，开启5分钟后方可关闭，以保证开启自启成功注册）

1.3与娃娃机的指令交互

1.4与指令服务器的交互

1.5与配置服务器的交互

1.6与局域网配置工具的交互

1.7断流重推

2018.1.3 add

1.8接受升级指令自动下载安装重启。

<br>安装时，如果配置好adb运行环境。可运行android-push_install.bat脚本一键自动覆盖安装并升级。

请留意更新并升级。如发现bug，欢迎及时反馈。


2.局域网配置工具

2.1查找局域网内的安卓板（启动了我们的应用的安卓板）

2.2与安卓板交互，实现局域网配置参数

2.3通用RTMP流低延迟播放功能--可以用来预览该娃娃机上正在推流的视频


3.公网配置工具

3.1与配置服务器的交互，实现公网配置自己帐号下的安卓板

3.2通用RTMP流低延迟播放功能


4.SimpleClient与配套的服务器使用。应配置为服务器所在的IP 以及7771端口。
